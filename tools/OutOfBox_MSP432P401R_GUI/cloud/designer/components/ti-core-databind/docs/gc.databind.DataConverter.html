Class: DataConverter

gc.databind. DataConverter

Constructor

new DataConverter()

Singleton Class to register data converters that will be used by the DataBinder to convert data between bindings of different types.

Methods

(static) convert(srcTypeopt, destType) → {*}

Method to convert an element of data from one data type to another.
Parameters:
Name Type Attributes Description
srcType string <optional>
 the type of the source that this converter is to be used on. If not supplied, then it will be the default converter for all source types if a specific one cannot be found.
destType string the type of the output value required from this conversion.
Returns:
- the converted data or undefined if no converter found.
