Class: AbstractBind

gc.databind. AbstractBind

Constructor

new AbstractBind()

Abstract class that implements IBind interface. Clients should not derive from this class directly. Instead, they need to derive from some of its derived classes: AbstractBindValue or AbstractBindAction.
Implements:
