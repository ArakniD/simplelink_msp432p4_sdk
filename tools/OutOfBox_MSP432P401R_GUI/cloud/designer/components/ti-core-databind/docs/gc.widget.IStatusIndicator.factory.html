gc.widget.IStatusIndicator. factory

Constructor

new factory()

Factory object used to retrieve status indicators for a given widget.

Methods

(static) get(widgetId) → {gc.widget.IStatusIndicator}

Factory method to create/retrieve status indicators for a particular widget identified by an id. The widget must be part of the top level web component in order for this to work. Nested widgets inside other web components will not be found.
Parameters:
Name Type Description
widgetId string the id of the widget to get a status indicator for.
Returns:
the status indicator created for this widget, or null if no widget found.
