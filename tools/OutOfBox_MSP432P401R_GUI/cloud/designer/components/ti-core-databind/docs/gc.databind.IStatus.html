Interface: IStatus

gc.databind. IStatus

Additional information like error or warning message that the model can provide to the widget. Most GUI Composer widgets show the status as a small error or warning icon to the top left of the widget.
Implementations:
  • gc.databind.AbstractStatus

Methods

getId() → {string}

Unique string that can be used to identify the type of error or warning in client's scripts. Once ids are defined they should not be changed because older scripts will expect the previously published values. Can be an empty string. It can be specific to a given model or shared between models.
Returns:
an unique string that can be used to identify the type of error in client's scripts.
Type
string

getType() → {gc.databind.StatusType}

There are two status types - error or warning. Different types are displayed with different visual clues in the widgets.

the type of this status: ERROR or WARNING.
