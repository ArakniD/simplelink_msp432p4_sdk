    48     *  provided by the Cache module delegates.
    49     *

    51     *  by the Cache module delegates.
    52     *
    53     *  Follow the link below to determine which Cache delegate is used
    54     *  for your Target/Device:
    55     *
    56     *      {@link ./../family/doc-files/delegates.html Delegate Mappings}

    58     *  @p(html)

    63     *
    64     *    <tr><th> Function                 </th><th>  Hwi   </th><th>  Swi   </th>
    65     *    <th>  Task  </th><th>  Main  </th><th>  Startup  </th></tr>
    66     *    <!--                                            -->
    67     *    <tr><td> {@link #disable} </td><td>   Y    </td><td>   Y    </td>
    68     *    <td>   Y    </td><td>   Y    </td><td>   Y    </td></tr>
    69     *    <tr><td> {@link #enable}  </td><td>   Y    </td><td>   Y    </td>
    70     *    <td>   Y    </td><td>   Y    </td><td>   Y    </td></tr>
    71     *    <tr><td> {@link #inv}     </td><td>   Y    </td><td>   Y    </td>

    75     *    <tr><td> {@link #wb}      </td><td>   Y    </td><td>   Y    </td>
    76     *    <td>   Y    </td><td>   Y    </td><td>   Y    </td></tr>
    77     *    <tr><td> {@link #wbInv}   </td><td>   Y    </td><td>   Y    </td>

    80     *       <ul>

    89     *             <li> During main().</li>
    90     *             <li> During BIOS.startupFxns.</li>
    91     *           </ul>
    92     *         <li> <b>Startup</b>: API is callable during any of these phases:</li>
    93     *           <ul>
    94     *             <li> During xdc.runtime.Startup.firstFxns.</li>
    95     *             <li> In your module startup before this module is started 
    96     *    (e.g. Cache_Module_startupDone() returns FALSE).</li>
    97     *           </ul>
    98     *       </ul>
