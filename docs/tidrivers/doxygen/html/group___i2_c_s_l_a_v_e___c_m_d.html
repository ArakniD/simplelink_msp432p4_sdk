Command Codes
I2CSlave_control command and status codes
Collaboration diagram for Command Codes:

I2C_CMD_* macros are general command codes for I2CSlave_control(). Not all I2CSlave driver implementations support these command codes.

I2CSLAVEMSP432_CMD_* macros are command codes only defined in the I2CSlaveMSP432.h driver implementation and need to:

#include <ti/drivers/i2cslave/I2CSlaveMSP432.h>
