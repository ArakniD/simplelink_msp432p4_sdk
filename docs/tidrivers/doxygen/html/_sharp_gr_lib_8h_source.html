79 // Translates a 24-bit RGB color to a display driver-specific color.
80 //
81 // \param c is the 24-bit RGB color. The least-significant byte is the blue
82 // channel, the next byte is the green channel, and the third byte is the red
83 // channel.
84 //
85 // This macro translates a 24-bit RGB color into a value that can be written
86 // into the display's frame buffer in order to reproduce that color, or the
87 // closest possible approximation of that color. This particular driver
88 // requires the 8-8-8 24 bit RGB color to convert into mono color
89 // 1 = White, 0 = Black
90 //
91 // \return Returns the display-driver specific color
92 //
93 //*****************************************************************************
