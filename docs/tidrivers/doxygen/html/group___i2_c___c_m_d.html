Command Codes
I2C_control command and status codes
Collaboration diagram for Command Codes:

I2C_CMD_* macros are general command codes for I2C_control(). Not all I2C driver implementations support these command codes.

I2CMSP432_CMD_* macros are command codes only defined in the I2CMSP432.h driver implementation and need to:

#include <ti/drivers/i2c/I2CMSP432.h>
