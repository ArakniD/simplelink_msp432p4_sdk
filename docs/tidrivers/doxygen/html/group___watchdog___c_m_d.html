Command Codes
Watchdog_control command and status codes
Collaboration diagram for Command Codes:

Watchdog_CMD_* macros are general command codes for Watchdog_control(). Not all Watchdog driver implementations support these command codes.

WatchdogMSP432_CMD_* macros are command codes only defined in the WatchdogMSP432.h driver implementation and need to:

#include <ti/drivers/watchdog/WatchdogMSP432.h>
