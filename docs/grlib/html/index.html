Graphics Library API Reference

The Texas Instruments Graphics Library is a set of graphics primitives for creating graphical user interfaces on microcontroller-based boards that have a graphical display. The graphical library consists of two building layers of functionality: the display driver layer, specific to the display in use; and the graphics primitives layer, which draws points, lines, rectangles, circles, text, and bitmap images.

The easiest way to get started is to click on the "Modules" tab above. This will provide an alphabetical list of graphics primitive available in the Graphics Library.

Graphics library comes prebuilt using the following compilers:

  • TI Code Generation Tools: ARM 18.1.0.LTS
  • IAR Code Generation Tools: ARM 8.20.2
  • GNU Code Generation Tools: gcc-arm-none-eabi-7-2017-q4-major
  • Keil uVision: 5.23
